# Glossary

| Term | Meaning |
| --- | --- |
| Admin socket | Local Unix domain socket used for mutating administrative operations. |
| Bucket | Shared object namespace that may also be exposed through file protocols. |
| Compatibility matrix | Test-backed statement of supported, partial, or unsupported behavior. |
| Mount SA | Identity used for mount-oriented workflows. |
| Phase 0 | Local anonymous development mode. |
| Phase 1 | Cluster bring-up stage. |
| Phase 2 | Authenticated operation with service accounts and policies. |
| Phase 3 | Production hardening stage. |
| Zero-config EC | Erasure-coding profile derived from cluster topology instead of a fixed startup choice. |
