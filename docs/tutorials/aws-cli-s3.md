# AWS CLI S3 Tutorial

Use this tutorial after the [quickstart](../start-here/quickstart.md). It focuses on the AWS CLI workflow for buckets and objects.

## Steps

1. Start GrainFS.
2. Upload an object with `aws s3 cp`.
3. List it with `aws s3 ls`.
4. Check supported S3 behavior in [S3 compatibility](../reference/s3-compatibility.md).
